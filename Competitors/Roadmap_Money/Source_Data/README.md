# Source Data

Raw research files that back up the profile. Most readers can ignore this folder — it's here so any claim in the profile can be traced back to evidence.

Files you'll find in here once research runs:

- `home.html`, `pricing.html`, `about.html` — saved copies of the pages
- `copy.txt` — all the text we pulled from the site
- `company_facts.json` — company info from LinkedIn, Crunchbase, etc.
- `design_details.json` — exact colors, fonts, and CSS measurements
- `meta.json` — when this was captured and from where
