# Context

Standing context the analyzer reads on every run. Edit these as the strategy sharpens.

- **`our_thesis.md`** — who we want to serve, what edge we think we have, what's off the table. The lens every competitor is read through.
- **`industry_frame.md`** — what we know about the wealth management / financial planning category. Patterns to look for, what "good" looks like, what to flag.

These files are inputs to `tools/analyze.ts`, not output documents. They shape the analysis without having to be re-stated in every prompt.
